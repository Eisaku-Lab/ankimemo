# ①課題名
暗記カードアプリ

## ②課題内容（どんな作品か）
暗記カードを作成・管理・学習できるWebアプリ。
学習したい問題と答えをカードに登録し、効率的に暗記学習を行うことサポートする。

### 主な機能
####カード作成機能
   - 問題と答えを入力してカードを作成
   - 作成したカードは自動的にlocalStorageに保存
   
####カード一覧表示機能
   - 作成した全てのカードを一覧で確認
   - 各カードには削除ボタンが付いており、不要なカードを個別に削除可能
   
####学習モード機能
   - 学習モードボタンをクリックすると、カードがシャッフル
   - カードをクリックすることで表面と裏面を切り替えて学習できる
   - 次のカードボタンで順番にカードを進めることができる
